Projeto acadêmico para avaliação final da disciplina de Computação Móvel,do curso de Análise e Desenvolvimento de Sistemas - UNITAU (2013).Trata-se de um aplicativo Android desenvolvido para tablets cujo objetivoé permitir o acesso a anúncios de classificados de imóveis na cidade de Taubaté - SP - Brasil;O instalador do aplicativo está na raiz do projeto (DroidImoveis.apk) e adocumentação (javadoc) pode ser acessada no diretório /doc.
